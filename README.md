# Ingenier-a-de-Software-Grupo-4-Backend
### Trabajo académico de Ingeniería de Software — Parte Backend

Este repositorio contiene el desarrollo del **backend** del proyecto académico para el curso de *Ingeniería de Software*.  
El sistema está implementado en **Java (JDK 21)**, utilizando **Spring Boot 3.5.7** como framework principal, y se conecta a una base de datos **MySQL alojada en AWS**.

---

## Tecnologías principales

| Tecnología | Versión | Descripción |
|-------------|----------|-------------|
| **Java** | 21 | Lenguaje de programación principal |
| **Spring Boot** | 3.5.7 | Framework para la creación del backend REST |
| **MySQL** | 8.x | Base de datos relacional (hospedada en AWS RDS) |
| **Maven** | Última estable | Sistema de construcción y gestión de dependencias |
| **JUnit / Spring Test** (Por ahora) | Incluido | Frameworks para pruebas unitarias e integradas |
